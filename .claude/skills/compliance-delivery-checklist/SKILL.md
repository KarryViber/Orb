# 对外交付审查清单

## When to Use
任何对外发送的内容（邮件、提案、报告、Slack 代言消息）发出前。

## 审查项
1. **受众确认** — 收件人是谁、角色、期望的正式程度
2. **边界检查** — 是否涉及未授权的承诺、价格、时间线
3. **事实核实** — 数据、名称、日期是否有来源支撑
4. **措辞审查** — 语气是否匹配受众（商务敬语/casual）
5. **语言检查** — 日语输出检查敬语层级、中文检查术语一致性
6. **脱敏** — 内部代号、API key、内部链接是否清除
7. **可追溯** — 发送后能否回溯到原始决策/数据源

## Notes
- 审查未通过的项必须明确标注，不能默认放行
- 紧急情况可缩减为项 2+3+6（边界+事实+脱敏）
